## 🌌 第一性原理 (First Principles)

Synapse 的设计不仅仅是关于“任务管理”，而是基于神经多样性心理学的深度洞察。我们拒绝将 ADHD 视为“缺陷”，而是将其视为一种“不同的认知操作系统”。

### 1. 时间坍缩 (Time Collapse)
*   **心理学原理**: ADHD 只有“现在”与“非现在”。线性时间感知的缺失（Time Blindness）导致了拖延和焦虑。
*   **Synapse 哲学**: 拒绝数字倒计时。使用**视觉沙漏**与**触觉反馈**，将抽象的“时间”转化为可感知的“物理实体”。让时间从“逝去的数字”变成“指尖的流沙”。

### 2. 多巴胺经济学 (Dopamine Economics)
*   **心理学原理**: ADHD 大脑缺乏持续的内在多巴胺供给，因此难以维持枯燥任务的动机。这并非“懒惰”，而是“神经递质饥荒”。
*   **Synapse 哲学**: 我们建立了一套**外部多巴胺补偿系统**。每一个原子动作的完成，都伴随着强烈的触觉震动和视觉烟花。我们将“执行”本身游戏化，用即时反馈填补动机缺口。

### 3. 认知卸载与外部脚手架 (Cognitive Offloading)
*   **心理学原理**: 工作记忆（Working Memory）的瓶颈使得多步骤任务（如“打扫房间”）在 ADHD 大脑中是一团无法解析的乱麻。
*   **Synapse 哲学**: **任务粉碎机 (Task Shredder)** 不是简单的列表，它是大脑的外部 RAM。它利用 AI 将模糊意图瞬间坍缩为不可再分的、2分钟内可执行的原子行动。

### 4. 关系去极化 (Relational Depolarization)
*   **心理学原理**: 在 ADHD 亲密关系中，很容易形成“焦虑的追逐者（非ADHD）”与“回避的疏远者（ADHD）”的极化模式，导致“亲子式”伴侣关系。
*   **Synapse 哲学**: **情感热力图**与**AI 代理催促**将“监督责任”从伴侣身上剥离，移交给算法。伴侣回归到“支持者”的位置，让爱不再被琐事磨损。

## 🚀 功能特性 (Features)

### 🏎️ 执行者 (The Executor - ADHDer)
*   **🧘 禅模式 (Calm Focus)**: 极简 UI，一次只显示一步，拒绝信息过载。
*   **⏳ 视觉沙漏 (Visual Timer)**: 像沙漏一样流逝的时间，而非焦虑的倒计时数字。
*   **🎉 多巴胺反馈 (Dopamine Hits)**: 完成任务时的强触觉反馈与视觉烟花。
*   **🆘 一键求助 (One-Tap Help)**: 降低羞耻感，快速请求伴侣介入（找东西、拆解任务）。

### 🧭 支持者 (The Supporter - Partner)
*   **✂️ 任务粉碎机 (Task Shredder)**: AI 辅助将模糊的大任务（如“收拾房间”）拆解为 <2分钟的原子步骤。
*   **📊 情感热力图 (Support Heatmap)**: 可视化你的付出，让隐形的情感劳动被看见。
*   **🎁 愿望商店 (Reward Shop)**: 积累“能量点”兑换实际奖励（按摩、免做家务卡等）。
*   **🤖 AI 代理催促 (Agent Mode)**: 让 AI 充当“坏人”去提醒，保护伴侣关系。

## 🏁 快速开始 (Getting Started)

### 前置要求
*   Node.js (LTS)
*   npm 或 yarn
*   Firebase 项目 (用于后端同步)
*   OpenAI API Key (可选，用于 AI 任务拆解)

### 安装步骤

1.  **克隆仓库**
    ```bash
    git clone https://github.com/shadowxx789/synapse.git
    cd synapse/synapse-app
    ```

2.  **安装依赖**
    ```bash
    npm install
    ```

3.  **配置环境变量**
    复制示例文件并填入你的 API Keys：
    ```bash
    cp .env.example .env
    ```
    *在 `.env` 中填入 Firebase 配置和 OpenAI Key。*

4.  **运行项目**
    ```bash
    # Web 端 (推荐调试 UI)
    npm run web

    # iOS 模拟器
    npm run ios

    # Android 模拟器
    npm run android
    ```
